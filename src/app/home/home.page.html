Welcome to Guess My Numbers Game

Let the Game Begin!

Try to guess the randomly generated number within a specified range.

Game Rules:

Easy Mode
(1-10)

1. The game will randomly generate a number between 1 and 10.
2. You have a maximum of 3 attempts to guess the correct number.
3. After each guess, the game will provide feedback (too high, too low, correct).
4. Your objective is to guess the correct number in as few attempts as possible.
5. If you don't guess the correct number within 3 attempts, you lose the game.

Medium Mode
(1-100)

1. The game will randomly generate a number between 1 and 100.
2. You have a maximum of 7 attempts to guess the correct number.
3. After each guess, the game will provide feedback (too high, too low, correct).
4. Your objective is to guess the correct number in as few attempts as possible.
5. If you don't guess the correct number within 7 attempts, you lose the game.

Hard Mode
(1-1000)

1. The game will randomly generate a number between 1 and 1000.
2. You have a maximum of 10 attempts to guess the correct number.
3. After each guess, the game will provide feedback (too high, too low, correct).
4. Your objective is to guess the correct number in as few attempts as possible.
5. If you don't guess the correct number within 10 attempts, you lose the game.
